# Homebrew Tap for mcleaner

This is the official Homebrew tap for [mcleaner](https://github.com/mpernia/mcleaner).

## Installation

```bash
brew install mpernia/mcleaner/mcleaner
```

## Documentation

See the [mcleaner repository](https://github.com/mpernia/mcleaner) for usage instructions.
